# Emergency Response System

A full-stack Emergency Response System built with the MERN stack.  
This project features **role-based dashboards**, **protected routes**, and a **responsive, mobile-friendly UI**.  
It simulates a real-world workflow: users request help, admins manage resources, drivers respond, and hospitals track patient flow.
